# 贡献者公约 / Contributor Covenant

## 我们的承诺

身为社区成员、贡献者及社区领袖，我们承诺让每个人参与本项目都不受骚扰，无论年龄、体型、可见或不可见的残障、族裔、性征、性别认同与表达、经验水平、教育程度、社会与经济地位、国籍、个人外貌、种族、种姓、肤色、宗教信仰、性倾向或性认同。

我们承诺以促进开放、欢迎、多元、包容、健康社区的方式行事与互动。

## 我们的准则

有助于营造正向社区环境的行为示例包括但不限于：

- 体谅他人，尊重不同观点。
- 给予并坦然接受建设性反馈。
- 承担责任，向受我们错误影响的人致歉，并从中学习。
- 着眼于社区整体，而非个人最佳利益。

不被接受的行为示例包括但不限于：

- 使用带有性暗示的语言或图像，以及任何形式的性挑逗或挑衅。
- 嘲讽、侮辱、人身或政治攻击。
- 任何公开或私下的骚扰。
- 未经明确许可，公布他人的私人信息（如真实地址、电子邮件地址）。
- 任何在专业场合下被认为不当的其他行为。

## 执行责任

社区领袖有责任阐明并落实可接受行为的准则，并对任何被认为不当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

社区领袖有权移除、编辑或拒绝与本公约不符的评论、提交、代码、wiki 编辑、Issue 及其他贡献，并在适当时说明理由。

## 适用范围

本公约适用于所有社区场合，并在个人代表项目或社区在公共空间中行事时同样适用。代表项目或社区的示例包括但不限于：使用官方电子邮件地址、通过官方社交媒体账户发帖、或作为指定代表出席线上或线下活动。

## 投诉与执行

如遇辱骂、骚扰或其他不可接受的行为，请通过 GitHub Issue（标注 `code-of-conduct`）或在仓库 README 列出的安全联系方式联系维护者。

所有投诉将被及时、公正地审查并调查。

所有社区领袖须尊重举报人的隐私与安全。

## 执行准则

社区领袖在判定违规行为时将参考以下社区影响指引：

### 1. 纠正

**社区影响**：使用不当语言或其他在社区中被视为不专业或不受欢迎的行为。

**后果**：来自社区领袖的私下书面警告，说明违规性质并解释行为为何不当。可要求公开致歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为违规。

**后果**：警告并附带后续行为的约束条件。在指定时间内不得与相关人员（包括公约执行者）有任何互动。包括避免在社区空间及外部渠道（如社交媒体）的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区准则，包括持续的不当行为。

**后果**：在指定时间内临时禁止与社区进行任何形式的互动或公开沟通。在此期间不得与相关人员（包括公约执行者）有任何公开或私下的互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出持续违反社区准则的行为模式，包括持续的不当行为、骚扰个人、攻击或贬低某类人群。

**后果**：永久禁止与社区进行任何形式的公开互动。
